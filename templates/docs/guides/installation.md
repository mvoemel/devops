# Installation Guide

Follow these steps to install the project:

## Prerequisites

- Node.js (>= 14.x)
- NPM (>= 6.x)

## Steps

1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/project-name.git
   ```
2. Navigate to the project directory:
   ```bash
   cd project-name
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
